

# Power Electronics Repository

## Overview

This repository contains my learning materials, designs, and simulation files for Power Electronics. It is organized to track my progress from basic concepts to advanced circuit designs and experiments.

## Repository Structure

```
simulations/       # Simulation files (LTSpice, MATLAB/Simulink, etc.)
designs/           # Circuit diagrams, schematics, and sketches
notes/             # Study notes, PDFs, and Markdown files
projects/          # Completed projects or exercises
datasheets/        # Component datasheets
```

## How to Use

1. Clone the repository:

```
git clone https://github.com/wogambula/power-electronics.git
```

2. Open the relevant folder:

* **simulations/** → open in your simulation software
* **designs/** → view or edit circuit diagrams
* **projects/** → follow step-by-step projects

3. Add new files and organize them in the correct folder as you learn.

## Best Practices

* Use meaningful file names (e.g., `buck_converter_LTSpice.asc`)
* Keep simulation and design files together for each project
* Include a short description for each project or experiment
* Use versioning for improved designs (v1, v2, final)

## Learning Goals

* Understand basic converter topologies: buck, boost, buck-boost
* Study switching devices: MOSFETs, IGBTs
* Learn PWM control and feedback loops
* Explore PCB design considerations for power electronics
